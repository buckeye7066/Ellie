## Runbook

### Dev commands
- Web: `pnpm dev:web`
- API/Gateway: `pnpm dev:server`
- One-click launcher: `pnpm ellie:start`
- Stop services: `pnpm ellie:stop`

### Packaged build
- Build renderer: `pnpm build:renderer`
- Build app: `pnpm build:electron`

### Environments
- Copy `env.example` to `.env` and adjust as needed.
